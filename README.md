# ColText

This include changes your symbols to different colors each time. The colors are randomized with each launch.

## How to use
`#include <ColText.hpp>`
`int main() {`
`    ColText("TERMCOLOR INCLUDE REQUIRED!");`
`    return 0;`
`}`
